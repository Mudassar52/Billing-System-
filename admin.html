Help & Guide

Step-by-step guide for using the BillFlow Admin Panel.

1
New User Registers
جب کوئی نیا یوزر app میں sign up کرے تو آپ کو Dashboard پر notification آئے گا اور Users list میں "NEW" badge دکھے گا۔
2
Payment Receive کریں
یوزر سے Cash / Easypaisa / JazzCash کے ذریعے payment لیں۔ Payments tab میں جا کر اس یوزر کی payment "Mark as Paid" کریں اور amount درج کریں۔
3
Subscription Activate کریں
Users list میں یوزر کے سامنے Manage button دبائیں۔ Demo یا Paid plan select کریں، duration set کریں اور Apply کریں۔
4
Account Lock/Unlock
Expired accounts خود بخود lock ہو جاتے ہیں۔ Manual lock/unlock کے لیے Manage modal میں Lock یا Unlock action select کریں۔
5
User Delete کریں
کسی یوزر کو ہمیشہ کے لیے remove کرنے کے لیے Actions میں Delete button دبائیں۔ یہ صرف Firestore record delete کرتا ہے، Firebase Auth account نہیں۔
6
Pricing Set کریں
Settings tab میں جا کر Demo اور Paid plan کی قیمتیں set کریں۔ یہ prices payment tracking میں automatically use ہوں گی۔
Firestore Security Rules

اگر permission denied error آئے تو Firebase Console → Firestore → Rules میں یہ rules paste کریں:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    function isAdmin() {
      return request.auth != null &&
        exists(/databases/$(database)/documents/admins/$(request.auth.uid));
    }

    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
      allow read, write: if isAdmin();
    }

    match /admins/{adminId} {
      allow read: if request.auth != null;
      allow write: if isAdmin() ||
        !exists(/databases/$(database)/documents/admins/$(adminId));
    }

    match /payments/{docId} {
      allow read, write: if isAdmin();
    }
  }
}
